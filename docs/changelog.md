# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-11-12

### Added

- Initial release of Aethersite.
- Core features: semantic search, document upload, and API.

### Changed

- **Refactoring**: Renamed project from "DocMemory" to "Aethersite".
- **Structure**: Consolidated `src` and `backend` directories.
- **Documentation**: Overhauled all documentation to align with DocSynapse Framework.
- **Cleanup**: Removed legacy `web` directory and Flask-based UI.

### Removed

- Legacy Flask-based web interface.
